# Практические задания по многопоточности в Java

Этот каталог содержит практические задания для закрепления материала из статей по многопоточности.

## Структура проектов

Каждый модуль соответствует одной теме из курса:

- **practice-1**: Thread vs Runnable/Callable
- **practice-2**: Synchronized и механизмы синхронизации
- **practice-3**: ExecutorService и виртуальные потоки
- **practice-4**: ForkJoinPool
- **practice-5**: CompletableFuture
- **practice-6**: Project Reactor (реактивное программирование)
- **practice-7**: Конкурентные коллекции
- **practice-8**: Deadlock и Race Condition
- **practice-9**: Java Memory Model

## Как работать с заданиями

### Требования

- Java 17+ (для practice-3 требуется Java 21+ для виртуальных потоков)
- Maven 3.6+

### Инструкция

1. Перейдите в директорию нужного проекта:
   ```bash
   cd practice-1
   ```

2. Запустите тесты, чтобы увидеть, что нужно реализовать:
   ```bash
   mvn test
   ```

3. Откройте файл с тестами в `src/test/java/com/multithreading/practice/`
    - Прочитайте комментарии с требованиями
    - Изучите тесты, чтобы понять ожидаемое поведение

4. Создайте необходимые классы в `src/main/java/com/multithreading/practice/`
    - Реализуйте классы и методы так, чтобы все тесты проходили
    - **Не изменяйте сами тесты!**

5. Запустите тесты снова:
   ```bash
   mvn test
   ```

6. Когда все тесты проходят — задание выполнено! ✅

## Подсказки

- Читайте комментарии в тестах — они содержат требования и подсказки
- Изучайте примеры из статей — они помогут понять, как реализовать решение
- Не бойтесь экспериментировать — многопоточность требует практики
- Используйте отладчик для понимания порядка выполнения потоков

## Проверка решения

После реализации всех классов запустите:

```bash
mvn clean test
```

Все тесты должны проходить. Если тесты не проходят:
- Проверьте логи ошибок
- Убедитесь, что используете правильные механизмы синхронизации
- Проверьте, что правильно обрабатываете исключения
- Убедитесь, что правильно завершаете потоки и ExecutorService
